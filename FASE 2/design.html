  • ETIQUETAS SEMANTICAS

    • Las etiquetas semánticas son una estructura que nos ayudan a definir las páginas web haciéndolas mejor deseadas por los buscadores.
    estas etiquetas se califican como semánticas cuando nos informa con exactitud de lo que trata su contenido, para así tener una mejor experiencia de navegación en sus paginas web.

  • CSS GRID

    • El Css Grid primero que todo es un modelo de maquetación CSS en base a una rejilla, el cual podemos utilizar para obtener muchos diseños diferentes, también es muy destacado por permitirnos dividir una página en áreas o regiones principales, nos permite definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de HTML.
